## Quick Start

```bash
# Clone the repository
git clone https://github.com/sagar0163/ascii-art-gallery.git

# Navigate to category
cd ascii-art-gallery/landscapes

# View ASCII art
cat mountain.txt
```

## Usage

### View Single Art
```bash
cat landscapes/mountain.txt
```

### List All Available
```bash
ls -la */
```

### Search for Art
```bash
find . -name "*.txt" | xargs grep -l "keyword"
```
