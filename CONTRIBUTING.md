# Contributing to ASCII Art Gallery

Welcome! We're excited you're interested in contributing.

## How to Contribute

### Adding New ASCII Art
1. Fork the repo
2. Create a new branch: `git checkout -b new-art-category`
3. Add your ASCII art to the appropriate folder
4. Commit: `git commit -m "Add new [category] ASCII art"`
5. Push to main: `git push origin main`

### Categories
- `landscapes/` - Nature scenes
- `objects/` - Everyday objects
- `characters/` - Characters and people
- `animals/` - Animal ASCII art
- `text/` - Text logos

### Guidelines
- Use monospace fonts for best rendering
- Keep width under 100 characters
- Test in terminal before submitting

## Commit Message Format
```
type: description

- Add feature/fix
- Another change
```

## Submit PR
Open a Pull Request to `main` branch.
